Shatter FAQ

Frequently asked questions

Notices

We conntect to the network sometimes.

We do not have explicit terms or policies; however, we do have a privacy statement since there are some features of Shatter for Blender that could pose a privacy or security risk due to using the network.

The Shatter for Blender developers do not currently and do not plan to collect any telemetry data from users.

We use open source software and distrubiute it with Shatter for Blender.

We use Python-RSA for cryptographically signing and verifying updates as well as requests for fetching them.

Common questions

Where can I learn how to use Shatter for Blender?

There are two official resources for learning Shatter for Blender: the Wiki and a short video series. You might find the videos easier to understand, though you are free to choose what you like.

Please keep in mind that the videos might be outdated, so if you see that something that looks different from the video, that is most likely why.

I am having a problem. Where can I get help?

Please join the Smash Hit Lab discord! There are many people who can help, so you are more likely to get a response.

Please do not try to contact the Shatter for Blender developers directly; we would like to help, but we don't have time to provide private support for everyone.

Does this addon update automatically?

Shatter for Blender will notify you if there is an update, but it will not install it by default. If you want updates to be installed automatically, go to Edit → Preferences → Add ons, search for "Smash Hit Tools", then enable the "Automatically install updates" option.

I found a bug. Where do I report it?

Please submit bugs to the issue tracker on GitHub.

Where can I submit a feature request?

Please submit feature requests to the issue tracker on GitHub.

Why is Shatter sometimes refered to as SHBT or "Smash Hit Blender Tools"?

Shatter for Blender used to be called Smash Hit Blender Tools when it was developed under the Smash Hit Lab. After leaving the Smash Hit Lab staff, the original developer continued the project under the name Shatter.

Other questions

Where can I find the Shatter for Blender GitHub page?

It is here: https://github.com/Shatter-Team/Shatter

Do you really think a Blender add-on is the best level editor for a mobile game?

For many games, we know it's probably not. From the start of the project, we did not intend to create the best possible editor; instead, we wanted to make some form of segment editing possible without resorting to editing text files, and more importantly, see what and what not to do in order to make a good editor.

From the original Shatter for Blender blog post:

"Really, this plugin is more about giving future flexibility and helping spot issues that would be helpful to know in the future."

Is there an Android version of Shatter?

Unfortunately, Shatter for Blender is not available for Android because Blender is not available for Android. Also, keep in mind that there are very few 3D level editors for games on mobile in general for good reasons — the experience would be painful!

If you really need to make levels on Android, you can try Carambola version 2021-10-17 (beta). Please note that Carambola does not receive updates anymore, and people are not likely to help you if you get stuck or run into issues.
