Smash Hit™ Blender® Tools is an addon for the 3D modeling software Blender that allows the creation of Smash Hit segments — the most fundamental part of a level.

What can it do?

Blender Tools is currently the most complete Segment Editor for Smash Hit. It can:

  • Create Smash Hit segments of varying complexity with boxes, obstacles, decals, power-ups and water.
  • Import other Smash Hit segments for learning and modifying.
  • Export LitMesh files alongside segments which look nearly indistinguishable from the official ones.
  • Make both compressed and uncompressed segment output.
  • And generally help create great Smash Hit segments!

Questions and Answers

Where can I learn how to use Blender Tools?

There are two official resources for learning Blender Tools: the Wiki and a short video series. You might find the videos easier to understand, though you are free to choose what you like.

Can I haz android apk link plz?

Unfortunately, Blender Tools is not available for Android because Blender is not available for Android. Also, keep in mind that there are very few 3D level editors for games on mobile in general for good reasons — the experience would be painful!

If you really need to make levels on Android, you can try Carambola 2021-10-17, though please note that Carambola does not receive updates anymore.

I found a bug. Where do I report it?

Please submit bugs to the issue tracker on GitHub.

Where can I submit a feature request?

Please submit feature requests to the issue tracker on GitHub.

Do you really think a Blender add-on is a good level editor for a mobile game?

No, at least not for Smash Hit. From the start of the project, we did not intend to create the best possible editor; instead, we wanted to make some form of segment editing possible without resorting to editing text files, and more importantly, see what and what not to do in order to make a good editor.

From the original Blender Tools blog post:

"Really, this plugin is more about giving future flexibility and helping spot issues that would be helpful to know in the future. I [Knot126] hope that this add-on, even if nobody really uses it, helps us make Smash Hit modding even easier for those that don't have much experience with the technical side of things."
